# Basic HTML Website

This is a simple project that demonstrates how to create and run a basic HTML website. The project contains only static HTML,  with no CSS and JavaScript files. https://roadmap.sh/projects/basic-html-website

## Getting Started

### Prerequisites

You only need a web browser (e.g. Chrome, Firefox, Edge, Safari) — no additional dependencies or installations are required.

### Running the Website Locally

1. **Clone this repository or download the source code:**
    ```bash
    git clone https://github.com/leductrung0110/basic-html-website.git
    ```
    or [Download ZIP](https://github.com/leductrung0110/basic-html-website/archive/refs/heads/main.zip) and extract it.

2. **Open the main HTML file:**
    - Navigate to the project folder.
    - Find the main file, usually `index.html`.
    - Double-click on `index.html` or right-click and choose “Open with” → your favorite browser.

That’s it! Your website should appear in the browser.

### Alternative: Running with a Simple Local Server (Optional)

If you want to test features that require a server (such as fetch/XHR requests), you can run a simple static server:

#### Using Python 3 (built-in):

```bash
cd path/to/basic-html-website
python -m http.server 8000
```
Then, open [http://localhost:8000](http://localhost:8000) in your browser.

#### Using Node.js (http-server):

If you have Node.js installed:
```bash
npm install -g http-server
cd path/to/basic-html-website
http-server
```
Then open the displayed local address.

## Project Structure

- `index.html` — Main entry point for the website.
- `style.css` (optional) — Styles for the site.
- `script.js` (optional) — JavaScript functionality.
